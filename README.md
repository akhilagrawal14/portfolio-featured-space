# Akhil Agrawal Portfolio

Personal portfolio website for Akhil Agrawal, built with Vite, React, TypeScript, shadcn-ui, and Tailwind CSS.

## Local Development

```powershell
npm.cmd install
npm.cmd run dev
```

The dev server is configured to run at:

```text
http://localhost:8080/
```

## Useful Commands

```powershell
npm.cmd run build
npm.cmd run lint
npm.cmd run preview
```

## Deployment

This is a static Vite app and can be deployed directly to Vercel.

- Framework preset: `Vite`
- Build command: `npm run build`
- Output directory: `dist`

The contact form uses a `mailto:` link, so it does not require a backend service or external form provider.
